# 순수함수란?

## 결론

순수함수란 동일한 인자를 전달하였을 때 항상 동일한 값을 반환하며, 사이드 이펙트가 없고 참조 투명한 함수를 의미합니다.

## 설명

- 사이드 이펙트란 함수 내의 실행으로 인해 함수 외부가 영향을 받는 것입니다.

- 참조 투명성이란 함수가 함수 외부의 영향을 받지 않는 것을 의미합니다.

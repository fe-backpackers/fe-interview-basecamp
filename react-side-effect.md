## 리액트에서 사이드 이펙트란?

## 결론

리액트에서 사이드 이펙트란,
리액트 컴포넌트가 화면에 렌더링된 이후에 비동기적인 로직등으로 처리되는 부수적인 효과들을 뜻합니다.

## 설명

컴퓨터 공학에서 사이드 이펙트란,
함수의 로컬 상태를 함수 외부에서 변경하는 경우, side effect가 발생했다고 합니다.

이것을 리액트에 적용해보면, 리액트 컴포넌트의 로컬 상태를 리액트 컴포넌트 외부에서 변경하는 경우가 사이드 이펙트라고 할 수 있습니다.

그래서 비동기 로직등은 useEffect를 사용하여 관리하는 것입니다.

## 요약

리액트에서의 사이드 이펙트는 컴퓨터 공학에서의 사이드 이펙트와 같은 개념으로,
함수 로컬 상태를 외부에서 조작해서 변경되는 것을 뜻한다.

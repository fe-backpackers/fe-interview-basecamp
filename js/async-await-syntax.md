# async-await 란?

## 결론

- `async-await`는 비동기 함수를 읽고 쓰기 편하게 만들어주는 syntatic sugar입니다.

## 설명

- ES8에 도입된 문법으로서, Promise 의 위에서 동작합니다.

- `async-await`를 사용하면 비동기 함수를 마치 동기 함수인 것처럼 작성할 수 있습니다.
